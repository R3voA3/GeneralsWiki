# **How to play with the lastest Release**

One of the many problems **C&C Generals and Zero Hour** have is the ecosystem fragmentation.
Over the past two decades there have been several tools and applications to help update and launch both the main game and its expansion.
The following guide will show you how to play **C&C Generals and Zero Hour** in 2026 with the help of **GenHub**.

## Game and Launcher Installation
1. Make sure to have both **C&C Generals and Zero Hour** installed. It does not matter whether you got it from Steam, Origin or good old CD.
2. Next, download and install the latest release of the [GenHub](https://github.com/community-outpost/GenHub/releases) application.
> [!NOTE]
This is a modern launcher application taking care of downloading the latest builds, patches and managing third party content such as mods for you.
Visit the [GeneralsHub Wiki](https://generalshub.netlify.app/wiki/) for a full list of features!
3. Now, launch the **GenHub** application.

## GenHub Launcher Setup
1. When starting the launcher for the first time make sure to press the **Scan** button in the top right.
It will scan your installations and download the official patches if necessary
2. Go to the download tab in the top center, select the arrow right to **TheSuperHackers** and select the release version you want from the drop down menu.
> [!NOTE]
Only go for an older release if the latest one causes issues for you.
3. You can also browse the other tiles such as **GeneralsOnline** for online game play or **GitHub** to download other community content
such as enhanced menus or control bars.

## Profil Setup
1. Now go back to the **Game Profiles** tab in the top left and press the **Add new Profile** button to create a new profile.
2. In the now open window, select **Installation** and select the installation you want to use, for example **Steam, Zero Hour**. Press the **+** button to add it to the current profile.
3. Next, press the **Executeable** button and select one of the executables. For example **SuperHackers - Zero Hour v2034..**.
4. Optionally, go to the **Profile Settings** and add a name, description and various other information for this profile. You can also add startup parameters here.
5. Last but not least, go to the **Game Options** and set up the settings you wish to use, such as game resolution, video quality and so on.
Most of the new options introduced with the **SuperHackers** patch are available here as well.
6. Now save the profile and press **Launch** to start the game. Enjoy!

## Camera Height Fix
1. Go the the **Downloads** tab in top center
2. Expand the **Community Outpost** section
3. Download **Camera Mod - General** or **Camera Mod - Zero Hour**
4. Add the mod to the profile of your choice
